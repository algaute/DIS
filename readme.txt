Installation 

1. Spring Tool Suite (https://spring.io/tools)
2. maven (https://maven.apache.org/)
3. java 8 jdk (http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
4. node.js V5.x.x (https://nodejs.org/en/)
5. npm 3.x.x (https://docs.npmjs.com/getting-started/installing-node)
6. mysql community server (http://dev.mysql.com/downloads/) version 5.7.13
7. optional : MySQL workbench and his tools

Spring
1. install DIS-Spring-data and DIS-Rest-service-cors 
	into Spring by "file->import->general->Import projet from folder"
2. setup the projects as a maven project

Angular
1. go to angular project folder DIS-angular and start npm by typing (npm start)
